# 🎭 Telegram-бот клуба «Мафия»

Бот для учёта фишек, ставок, бонусов и статистики игроков клуба.

---

## 📁 Структура проекта

```
mafia_bot/
├── app/
│   ├── main.py                  <- Точка входа. Запускать отсюда
│   ├── config.py                <- Настройки (читает .env файл)
│   ├── database/
│   │   ├── models.py            <- Структура таблиц базы данных
│   │   └── queries.py           <- Все операции с базой данных
│   ├── handlers/
│   │   ├── common.py            <- /start, главное меню, помощь
│   │   ├── player.py            <- Профиль, баланс, рейтинг
│   │   ├── admin.py             <- Функции администратора
│   │   ├── bets.py              <- Ставки
│   │   └── diary.py             <- Дневник Ребекки Найт
│   ├── keyboards/
│   │   └── main_kb.py           <- Все кнопки и меню
│   ├── services/
│   │   └── sheets_service.py    <- Интеграция с Google Sheets
│   └── utils/
│       ├── formatters.py        <- Форматирование текста
│       └── states.py            <- Состояния диалогов (FSM)
├── seed_data.py                 <- Тестовые данные
├── requirements.txt             <- Список библиотек
├── .env.example                 <- Пример настроек
└── README.md                    <- Этот файл
```

# 🚀 ПОШАГОВАЯ ИНСТРУКЦИЯ

## Шаг 1: Установка Python

### Windows:
1. Зайди на https://www.python.org/downloads/
2. Нажми "Download Python 3.12..."
3. Запусти скачанный файл
4. ВАЖНО: поставь галочку "Add Python to PATH"
5. Нажми "Install Now"

Проверь установку — открой Командную строку (Win+R, cmd) и введи:
```
python --version
```
Должно написать: Python 3.12.x

### macOS:
```bash
brew install python3
```

### Linux:
```bash
sudo apt update && sudo apt install python3 python3-pip
```

---

## Шаг 2: Установка библиотек

Открой командную строку в папке mafia_bot:
```bash
pip install -r requirements.txt
```

---

## Шаг 3: Создание бота через BotFather

1. Открой Telegram, найди @BotFather
2. Напиши /newbot
3. Введи название (например: Клуб Мафия)
4. Введи username — должен кончаться на bot (например: mafia_club_bot)
5. BotFather пришлёт токен вида: 1234567890:AAHdqTcvCH1vGWJxfSeofSs4tDbC6bAHpB8
6. Сохрани токен!

---

## Шаг 4: Узнай свой Telegram ID

1. Найди бота @userinfobot в Telegram
2. Напиши ему что угодно
3. Он ответит твоим ID — набор цифр, например: 123456789

---

## Шаг 5: Настройка .env файла

1. Скопируй файл .env.example, назови копию .env
2. Открой .env и заполни:

```
BOT_TOKEN=1234567890:AAHdqTcvCH1vGWJxfSeofSs4tDbC6bAHpB8
ADMIN_IDS=123456789
GOOGLE_CREDENTIALS_FILE=credentials.json
GOOGLE_SHEET_ID=вставь_id_таблицы
GOOGLE_SHEET_NAME=Статистика
DATABASE_PATH=mafia_club.db
```

Несколько администраторов: ADMIN_IDS=111111,222222

## Шаг 6: Подключение Google Sheets

### 6.1 Создать проект Google Cloud
1. Зайди на https://console.cloud.google.com/
2. Вверху нажми список проектов -> "Новый проект" -> назови -> "Создать"

### 6.2 Включить API
1. Левое меню -> "API и сервисы" -> "Библиотека"
2. Найди "Google Sheets API" -> нажми -> "Включить"
3. Найди "Google Drive API" -> нажми -> "Включить"

### 6.3 Создать сервисный аккаунт
1. Левое меню -> "API и сервисы" -> "Учётные данные"
2. "+ Создать учётные данные" -> "Сервисный аккаунт"
3. Имя: mafia-bot -> "Создать и продолжить" -> "Готово"

### 6.4 Скачать JSON-ключ
1. Нажми на созданный аккаунт
2. Вкладка "Ключи" -> "Добавить ключ" -> "Создать" -> JSON
3. Файл скачается. Переименуй в credentials.json
4. Положи credentials.json в папку mafia_bot/app/

### 6.5 Открыть доступ к таблице
1. Открой credentials.json в блокноте
2. Найди поле "client_email" — скопируй адрес
   (похоже на: mafia-bot@project.iam.gserviceaccount.com)
3. Открой свою Google-таблицу
4. Нажми "Поделиться" -> вставь email -> "Просматривать" -> "Готово"

### 6.6 Узнать ID таблицы
В адресной строке браузера:
https://docs.google.com/spreadsheets/d/ВОТ_ЭТОТ_ID/edit
Вставь этот ID в .env в поле GOOGLE_SHEET_ID

### 6.7 Структура таблицы
Первая строка — заголовки. Поддерживаемые названия колонок:

  player_id     — уникальный ID игрока (обязательно)
  nickname      — ник игрока (обязательно)
  games_played  — количество игр
  rating        — рейтинг
  status        — статус (Новичок, Мастер...)
  rank_position — место в рейтинге
  wins          — победы
  survived      — выжил
  city_wins     — победы города
  mafia_wins    — победы мафии

## Шаг 8: Тестовые данные (необязательно)

Чтобы поиграться без Google Sheets — запусти:
```bash
python seed_data.py
```
Создаст 5 тестовых игроков.

## Шаг 9: Привязать игрока к Telegram

Чтобы пользователь видел свой профиль:
1. Пользователь пишет боту /start
2. Узнай его Telegram ID (через @userinfobot)
3. Зайди как администратор
4. Нажми "Привязать игрока"
5. Введи Telegram ID пользователя
6. Введи player_id из таблицы (например: P001)

---

## Шаг 10: Как назначить администратора

Способ 1 (рекомендуется): добавь ID в .env файл:
  ADMIN_IDS=123456789
Перезапусти бота.

Способ 2: через базу данных (SQLite Browser или командная строка):
  UPDATE users SET is_admin = 1 WHERE telegram_id = 123456789;

## Частые проблемы

Ошибка: ModuleNotFoundError: No module named 'aiogram'
Решение: запусти pip install -r requirements.txt

Ошибка: Токен бота не задан
Решение: убедись что файл .env существует и токен правильно заполнен

Ошибка: FileNotFoundError: credentials.json
Решение: положи credentials.json в папку app/ и проверь имя в .env

Ошибка: SpreadsheetNotFound
Решение: проверь ID таблицы в .env и доступ сервисного аккаунта

Бот не отвечает на кнопки профиля:
Решение: нужна привязка игрока (шаг 9 выше)
